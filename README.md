# Tes Praktikum Frontend Developer
Halo, ini adalah repositori untuk tes praktikum Web Developer PT. Buat Usaha Digital Indonesia. Nah untuk joblistnya adalah merancang dan membangun tampilan website serapi dan sebagus mungkin.

Dan kriterianya adalah
1. Buatlah tampilan website berdasarkan kerangka awal dari berkas `index.html` itu.
2. Silahkan kustomisasi dengan selera anda dan boleh menggunakan bantuan CSS Framework (seperti bootstrap, tailwindcss, bulma dan lainnya) supaya lebih mudah.
3. Buat branch baru pada repositori ini dan push dengan nama branch yang sudah dibuat.

Selamat Mengerjakan, Semoga Berhasil!
